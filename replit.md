# COP3530 - Project 2

## Overview
This is a C++ educational project focusing on scheduler implementation and GitHub collaboration. The project is a template repository for learning data structures, algorithms, and version control best practices.

## Current State
- **Language**: C++17
- **Build System**: Makefile
- **Compiler**: Clang++
- **Main Entry Point**: src/heap.cpp (currently contains a basic setup check)

## Project Structure
```
├── src/
│   ├── heap.cpp       - Main entry point with setup check
│   ├── scheduler.cpp  - Scheduler implementation (empty)
│   ├── edf.cpp        - EDF algorithm implementation (empty)
│   └── *.h files      - Header files (empty)
├── resources/         - Documentation resources
├── data/              - Data files
├── docs/              - Documentation
└── visuals/           - Visual resources
```

## Building and Running
- **Build**: `make` or `make all`
- **Run**: `make run` or `./heap`
- **Clean**: `make clean`
